Hello, With the help of this plugin you allow your users to show the most watched videos on youtube in the current week.

== Description ==

Ever wondered which are the most watched videos on youtube this week??


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `youtube-most-watchedvideos-this-week` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Widgets and add this plug in in the sidebar and you are all set.
